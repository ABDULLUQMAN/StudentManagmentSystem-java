# StudentManagmentSystem-java
This is a simple Student Management System developed using Java Servlets, HTML, JDBC, and MySQL.

## Features
- Register new students
- Update student details
- Delete student records
- Display student information

## Technologies Used
- Java
- Servlets
- HTML
- CSS
- JDBC
- MySQL
- Apache Tomcat
- Eclipse IDE

## Project Structure
The project contains Java servlet files for backend logic and HTML files for frontend interface.
